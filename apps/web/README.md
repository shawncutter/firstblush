# FirstBlush Web MVP Demo

Web-first demo client for FirstBlush.

## Run

From this directory:

```bash
npm install
npm run dev
```

Default URL:
- `http://localhost:13110`

Expected API:
- `http://localhost:18110/v1`

Override API base:

```bash
VITE_API_BASE=http://localhost:18110/v1 npm run dev
```
